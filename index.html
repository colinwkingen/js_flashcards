JavaScript (JS)

most common scripting language used in the web.

Operators

symbols that conduct mathmatical operations like division or addition.

Variables

container with a name used to hold any value like a string or a number or a boolean.

Variable naming conventions

JavaScript uses camel case where first word is not capitalized and any subsequent words have their first letter capitalized.

Functions

A piece of code with input and output that perform a specific task.

Methods

A procedure done on an object or piece of data.

Arguments

The input values of function.

Parameters

The input variables of function.

Return

Returns values from functions.

Chaining methods

Methods that are used together one after the other(ie. method1.method2).

Strings

Series of letters and/or numbers that don't have numeric value.

Booleans

true/false

Undefined

Undefined value/ value that is not present.

NaN

Not a number. NaN's data type is a number.

Escape

Deprecated function that converts special characters into hexadecimal code.

Alert

Opens a popup window with your alert message.

Comments

Piece of writting that is in the code but not executed. Used to assist in understanding the code.

JQuery

Library of JS methods and functions.

Attributes

Specification that defines a property of an object.
